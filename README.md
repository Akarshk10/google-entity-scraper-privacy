# Privacy Policy – Google Entity Scraper by Akarsh

Google Entity Scraper by Akarsh does not collect, store, transmit, or share any personal or sensitive user data.

The extension operates entirely within the user’s browser and only reads visible Google Search result page elements when explicitly triggered by the user. All processing happens locally, and no information is sent to external servers or third parties.

The extension does not collect:
- Personally identifiable information
- Health information
- Financial or payment information
- Authentication data
- Personal communications
- Location data
- Web browsing history
- User activity data

The extension does not use analytics, tracking technologies, cookies, or remote servers.
